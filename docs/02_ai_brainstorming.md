# AI Brainstorming

Initial product options:

1. SkillBridge AI: a simple dataset-first skill-gap mapper.
2. PathForge AI: a broader transition planner with profile evidence, market validation, resource discovery, and process documentation.

## Decision

Use PathForge AI as the product name and keep the SkillBridge-style deterministic scoring core.

## Chosen MVP

- Current role selector
- Target role selector
- Resume or CV text/PDF intake
- Typed skill tags
- GitHub profile or repository scan
- Portfolio, LinkedIn, publication, and extra link intake
- Personalized compatibility score
- Transferable skills
- Prioritized skill gaps
- Explainability panel
- Alternative career routes
- 30/60/90 day learning plan
- Exa learning resources
- Apify job postings
- Optional login for saved roadmaps
- Repository-level process evidence in `docs/` and `logs/`

## Scope Removed From The Main UI

The dashboard no longer shows the 3-person team structure, process-score notes, dataset sidebar, or internal recommendation pipeline. Those belong in the repository and submission materials, not in the user's career-analysis workflow.
